# YukiSearch

---

## 簡介

### 整合多網站 動畫搜尋與管理

---

## 功能

- **一鍵查詢**：支持動畫瘋、囧次元、Anime1、櫻花等網站
- **一鍵導向網站**：不耗本地空間
- **統一管理**：不同網站的動漫 一起收藏
- **標籤管理**：為喜歡的動畫添加自定義標籤
- **觀看進度紀錄**：看到哪裡? 軟體幫你記
